Installation

order to get Piep running locally on your machine, it is required that you have the following things installed on your machine:

  • Git
  • Node
  • Grunt

Up and Running

you have installed git, node, and grunt

  • Clone the project from github

    clone git@github.com:piep/piep.git

  • Install dependencies

    install

    you run into errors, you may need to change permissions.

  • Start the project on a local server
