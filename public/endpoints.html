Piep API Endpoints

Objectives

are student goal statements.

Resources

are exercises, videos, blog posts, tutorials, and other material that would help a student reach an objectives.

Curated Lists

lists are lists of resources and objectives that center around a specifc category or topic that a student would want to learn. Members of the community build these lists to make it easier for students to find the best related content.

About piep!

is a package manager for educational materials related to teaching programming and computer science. Creating good content for a class is the biggest bottleneck in developing good educational programs in these fields, partially because the field moves so quickly. Piep.io aims to harness the power of the community to alleviate this bottleneck and produce relevant up-to-date open source content for teachers.
